信息繁杂，还需分而化之

YiCH 9月 28, 2016
  • Note
  • Soft
  • 分享到微博
  • 分享到 Twitter
  • 分享到 Facebook
  • 分享到 Google+

如今大部分人几乎脱离纸和笔，收集、记录信息都已数字化。同时，这个时代的信息量又是如此巨大，倘佯在信息之海，各种信息若只是借助单一的应用小船拖载，不仅杂乱、低效，更重要的是若小船翻了，上面的信息也会湮灭。所以我们不仅要学会将繁杂的信息分类记录，我们还要对自己的信息数据安全负责。

如何保证信息的记录易整理且数据安全？不少读者可能烦恼过这个问题。“不要将鸡蛋放在一个篮子里”，分而化之，这是最简单也是最安全的信息管理模式。我结合自身分化解决信息问题的经验给大家一些建议，其中会牵涉到比较多的软件，我会简单介绍各软件在我的信息系统中的作用，并会推荐类似应用给大家提供更多选择，希望大家会有耐心浏览到最后。

声明：除了 Workflowy 邀请链接 和幕布邀请链接 外，以下软件和链接无任何利益相关，评测我会尽量选择 ifanr 的文章。

一、信息收集

网络世界里信息丰富，也是我们主要的信息来源。我们浏览过的信息不可能全部用大脑记忆，快捷有效地采集这些信息是提高信息消化效率的主要途经。

曾写过一篇专门介绍网络信息收集应用的小文章，点击阅读。

这类的应用主要分两类，共有如图四个。

1. Pocket

Pocket 无论是移动版还是网页版，对网页再排版效果都特别好，有 Chrome 一键保存扩展及离线阅读应用，配合移动端应用，使用很方便。我用来保存有深度的人文类、生活类文章，包括短篇小说、散文、哲学、科学等等。

ifanr 评测文章：稍候阅读随身行 – Pocket 中文版。

2. Diigo

Diigo 的 Chrome 插件 Diigo Web Collector 不仅可以收藏网页，还有高亮网页文字，添加笔记和截屏保存等功能。结合 Diigo 笔记工具 Quick Note(需到 Chrome 应用商店下载) 和移动端应用可以轻松管理。这款软件因为可以添加注记，我用来保存学习和工作的文章，重点部分轻松保存。

Chrome插件网测评：Chrome书签、注释、截图、标记插件：Diigo Web Collector。

3. Raindrop.io

在 Instapaper 和 Readerbality (已经停止服务) 停滞不前时，只有 Raindrop.io 还在继续开发以抗衡 Pocket 的不断壮大。其 Chrome 插件有侧边操作栏，不用登录网页可以就可以管理收藏列表和进行设置操作，也提供了移动端应用。这样就比较适合我这种喜欢折腾软件的人，用来保存教程类的文章再方便不过了。不过这个软件有时候保存的速度稍慢，也不能单独收集文字或者图片，只能收集整个网页。

ifanr 评测：你要的完美云书签，可能就是它 – Raindrop.io 。

4. 花瓣

花瓣网提供浏览器插件，能够在网页批量、截图采集图片到花瓣网，是公开的。另外还有书签栏采集工具，以供不能使用插件的浏览器使用。花瓣移动端应用也是在停更几年后重新启动。这个我主要用来收集图片。

同类应用推荐

  • 方片收集 全能便捷的收集应用，之前很喜欢用这个，但是后来有一次登陆的帐号竟然被替换了(官方给的解释是程序员后台操作时不小心把自己的帐号全局了)，就抛弃了。确实好用。
  • 收趣 只能在移动端阅读收藏的文章。
  • Candy 和 Rraindrop.io 类似，无移动端应用 iOS 端已发布测试版， Android 后续会发布。

二、信息编写

生活中的个人信息、想法的记录，是自己最重要的隐私，这会关系到财产安全甚至人身安全，这类应用选择要慎重，主要以安全易用的为主。

1. 临时信息

临时信息顾名思义就是没有个人信息，不需要长久保存，只是短时间需要使用，比如某个陌生人联系方式、约会地点、购物清单等。这类选择就比较轻松，因为无重要个人信息，且可不用联网，那么个人信息就不容易被他人获取。

(1) 随笔记

随笔记(GNotes 的中国大陆版)，很轻量的笔记应用，选择这个主要是考虑到临时的信息很杂，各种内容都有。该软件移动端使用方便，清单、提醒、照片、录音、手指画和附件都可以加入笔记，可以同步到帐号或者Gmail，访问Web也可以浏览笔记，Chrome 有记录笔记插件。超过一定字数后会卡顿。

(2) 文本编辑器

这个在桌面使用，新建一个临时信息.txt 文件放在桌面，需要记录时直接打开写入，整理也很方便。

同类应用推荐

  • ColorNote 制作待办事项和购物清单，可以使用密码来保护便签， 支持备份到SD卡， 支持在Google号备份和同步。
  • Parchi 微软出品，不能同步。
  • Zoho notebook 出自”全球第一大在线软件提供商”Zoho，功能全面，界面美观，手势操作很华丽便捷。
  • 舒心录 个人产品，支持Markdown语法、单条加密和本地及网盘备份。
  • CatMemoNote PC桌面上简洁好用的记事软件，是日本人作品，个人汉化，类似的还有很多，Minipad2，Treepad，Mempad等，可以看看相关推荐 ，不过其中部分停止更新了，需要的可以自己搜索下载。

2. 非临时信息

这类笔记比较重要，需要长期保存，有比较重要的隐私，比如不常用易忘的帐号、注册信息、纪念日、地址等。

(1) Simplenote

Simplenote 是与博客平台 Wordpress 同厂出品，只能记录纯文本，支持 Markdown 语法，各种平台都支持，真正的全平台，同步速度也相当快，几乎是瞬时。除了记录信息外，平常手机上的写作我也在这里进行。桌面端比较重，可以考虑第三方应用 Notation，同样可以双向即时同步。

异次元评测:Simplenote - 简约极致、轻量高效的云同步纯文本笔记软件 (支持 Markdown 写作)。

(2) Workflowy(邀请链接)

Workflowy 是一款简洁而不简单的纯文本记录应用，每条 Iterm 可无限制向下建立分支，用来记录头脑风暴相当合适，还可以用作 Todo 来管理未完成工作，也可以用来编写提纲。通过 # 和 @ 建立标签轻松分类管理全局 Iterms，支持团队协作。Chrome 离线应用、Web 版和移动端都非常容易上手。这款优秀的应用我是用来写工作日志和文章提纲，结合幕布可以制作思维导图，利用 Chrome 插件 Workflowy for coder 可以支持 Markdown ，结合 Stylish 还可以可插入图片，移动端结合 Orgzly 可以设置提醒。

ifanr 评测：WorkFlowy：整理你的思绪
知乎使用经验：如何玩转 WorkFlowy？

(3) OneNote

这个是我整个笔记系统里面唯一的重量级应用，因为有些东西必须要用到大厂商相对可靠、安全的服务，比如说敏感个人信息、日记、重要超长篇笔记，并且可以配合 Office 和 Onedrive 使用，用来知识整理还是比较方便。这个应用在手机上可能比较慢，我一般用随笔记编写，然后利用分享功能保存到这里，既避免使用臃肿的软件本体，又能安全保存数据。

ifanr 使用经验： OneNote 深度教学：这 17 个功能你必须知道。

同类应用推荐

  • 幕布 基本功能与 Workflowy 相似，可以生成简单的思维导图，该文章内的树状图都是在这里生成。
  • Dynalist 等 Chrome 和 Android 应用出来了就换这个。“我们最初想做 Dynalist 就是因为 WorkFlowy 不能满足我们的需要……现在 WorkFlowy 有的基本都有了”该网站Dynalist 的创始人之一如是说——来自知乎Dynalist 与 WorkFlowy 主要的区别，或者说改进在什么地方？的回答。
  • Evernote 国内版是大家熟悉的“印象笔记”，全能型，第三方软件也很多，如果不喜欢大部头可以使用轻量级的移动客户端 Evermemo 或者 Purenote，在特定的文件内记录笔记可以双向同步。

3. 非常重要的隐私

Box

对于正值青春年少的我们，浏览网页时可能会有一些特别的东西需要保存，比如下载链接、图片，或者不愿意被人看到，或者根本就不能被任何人看到。这部分内容我就用了国内比较小众的 Box 网盘，结合 ShareX 桌面应用和 Chrome 插件可以将图片或者文字一步上传到 Box，移动端可以用 Box 官方应用，也可以用 ES 文件浏览器也可以上传和浏览盘内文件。

同类应用推荐

  • Yandex 网盘 俄罗斯著名搜索引擎出品，传输速度没有 Box 快。
  • Drobox 网盘 对，我们这部分就是要选择小众的，比较难用的，才能杜绝别人来偷窥隐私。

三、信息发布

信息的发布对于我们普通人就是写博客或者微博了。

1. Farbox

博客我选择了 Farbox，因为浏览量比较少，买了30元五年的服务，每个月100M，比 Wordpress 之类需要搭建网站的更简单上手。配套有编辑器 FarBox Editor，支持 Markdown 写作，登陆账号后写的文章自动发布到博客。可以同步到 Drobox。

Farbox 可以不用图床直接插入图片，但是博客内容加载比我用的图床 Tinypic 慢，Tinypic 也可以结合 ShareX 使用，截图、图片上传和外链复制都很方便。

36Kr评测：[博客还有另外一种极简的可能性，叫 FarBox。] (http://36kr.com/p/217779.html)

同类应用推荐

  • 简书 比较年轻化的一个写作与阅读社区，网页 Markdown 支持相当好，写作与阅读体验都很棒。
  • Wordpress 著名博客平台，要自己研究折腾，模板比 Farbox 多。

2. 微博

这个大家都会使用，就不多说了。选择用来发布个人心情，有趣、有益、简单的事情。
